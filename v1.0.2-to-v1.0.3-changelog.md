# 墨读小说 — 版本 1.0.2 → 1.0.3 提升总结
# MoRead Novel — v1.0.2 → v1.0.3 Upgrade Summary

**发布日期 / Release date**：2026-06-27  
**版本标签 / Version tag**：`v1.0.3`（commit `31c9ccf`）

---

## 一、阅读器核心重构 / Reader Core Refactoring

### 1.1 DOM 测量分页重写 / DOM-Measured Pagination Rewrite

- **替换**了旧的 CSS Columns 分页方案，改用 DOM 测量分页
- 隐藏容器逐段测高 → 贪心切分翻页
- **彻底消除**单页裁切偏移 bug，翻页精确度大幅提升

> **EN**: Replaced the old CSS Columns pagination with DOM-measured pagination. Hidden container measures each segment's height → greedy split into pages. Completely eliminated single-page clipping offset bugs — pagination accuracy greatly improved.

### 1.2 小说阅读器后端重写 / Novel Reader Backend Rewrite

- 内存缓存 + 后端分页 + 精确进度追踪
- 章节预取缓存、同步切换、无过渡动画，翻页性能和流畅度提升

> **EN**: In-memory cache + backend pagination + precise progress tracking. Chapter pre-fetch cache, synchronous switching, no transition animations — improved page-turn performance and smoothness.

### 1.3 底部栏全面重构 / Full Bottom Bar Restructure

| 改进项 / Improvement | 说明 / Description |
|--------|------|
| SVG 图标底部栏 | 替代旧版右侧设置抽屉，UI 更直观 / Replaced the old right-side settings drawer — cleaner, more intuitive UI |
| 翻页/滚动模式切换 | 新增 FlipPageIcon 按钮 / New FlipPageIcon button added |
| 单页/双页切换 | 滚动模式下按钮半透明禁用 / Semi-transparent & disabled in scroll mode |
| 主题颜色移入底部栏 | 无需打开菜单即可切换主题 / Switch themes without opening a menu |
| 行距/宽度改为滑块 | 更精细调节，操作更便捷 / Finer-grained adjustment, easier to use |
| Popover 关闭判定 | 改善弹出面板的交互体验 / Improved popover interaction logic |
| **移除自动翻页** | `autoFlipInterval` 相关代码完全移除，简化逻辑 / All `autoFlipInterval` code removed — logic simplified |

### 1.4 顶部栏优化 / Top Bar Optimization

- 移除顶部栏"目录"按钮 / Removed the "Table of Contents" button
- 移除顶部栏单页/双页切换按钮 / Removed single/double page toggle buttons
- WindowControls（全屏/关闭/小窗）**常驻展示**，移除 hover 折叠 / WindowControls **always visible**, removed hover-to-reveal
- 齿轮设置图标改为常驻，不再需要点击箭头展开 / Settings gear icon now always visible

## 三、正文搜索（全新功能）/ In-Text Search (New Feature)

Chapter range restriction: collapsible panel + start/end input + one-click "current chapter" button. Range optimization — left boundary starts at 1, right boundary no limit. Collapsed by default.

## 四、翻页体验优化 / Page-Turning Experience Optimization

### 往回翻章修复 / Fix: Going Back to Previous Chapter

- **Page Mode**：翻页模式下回翻章跳转到**上一章最后一页**，而非第一页 / Lands on the **last page**, not the first
- **Scroll Mode**：滚动模式下回翻章自动滚到底部 / Auto-scrolls to the bottom
- 新增 `goToEndRef` 和 `scrollToEndRef` 双机制 / Dual mechanism
- 单页模式直接使用 `pages.length - 1`
- 双页模式取最后一个偶数索引作为左页 / Last even index as left page
- 翻页操作增加边界检测与提示 / Page-turn boundary detection and toast hint
- 侧栏当前章节自动滚动定位 / Sidebar auto-scrolls to current chapter
