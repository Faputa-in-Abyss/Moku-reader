# 墨读 — 文件功能索引

所有前端代码在 `novel-reader/src/`，后端在 `novel-reader/src-tauri/src/`。

## 布局/框架

| 文件 | 功能 |
|---|---|
| `App.tsx` | 根入口/路由：根据 viewMode/reading/mangaReading 切换 Library/Reader/MangaReader |
| `store.ts` | Zustand 全局状态：小说/漫画数据、阅读状态、主题、所有设置偏好 |
| `styles/global.css` | 全局样式：CSS 变量、亮/暗主题色、毛玻璃、动画、响应式 |
| `main.tsx` | ReactDOM 挂载点 |

## 小说模块

| 文件 | 功能 |
|---|---|
| `components/Library.tsx` | 书架：小说列表、批量选择、图标更换、重命名、收藏、排序（最近阅读）、瀑布流卡片 |
| `components/Reader.tsx` | 阅读器容器：组装顶部栏、底部栏、侧栏、章节列表、hooks、右键菜单、段落书签；滚轮翻页限速 wheelLock=30ms（行 389-391） |
| `components/BottomBar.tsx` | 底部设置栏：字号/行高/字间距/段缩进/对齐/字体/页宽调节、翻页/单双页/滚动模式切换 |
| `components/Header.tsx` | 顶部标题栏：主题切换、设置面板、视图切换（小说/漫画/粒子） |
| `components/ChapterList.tsx` | 章节目录侧栏：目录树、书签跳转、正文搜索、搜索范围限定 |
| `components/PageRenderer.tsx` | 段落渲染：按段落数组渲染文本、段落级书签下划线、文字样式参数 |
| `components/WindowControls.tsx` | 窗口控制按钮：最小化/最大化/关闭、hover 折叠或常驻 |
| `components/SidebarHandle.tsx` | 侧栏吸附把手：边缘提示脉冲动画 |
| `hooks/usePagination.ts` | DOM 测量分页：隐藏容器逐段测高 → 贪心切分页，翻页=setState(index) |
| `hooks/useWheelHandler.ts` | 滚轮翻页/防抖：THRESHOLD=120px 累加触发、ANIMATION_DURATION=30ms 动画锁；仅管跨章边界滚轮（非普通翻页） |
| `hooks/useReaderKeyboard.ts` | 键盘快捷键：方向键翻页、字号加减、快捷键录制 |
| `hooks/useChapterLoader.ts` | 章节加载：调用 Rust 后端加载章节文本、构建段落列表、预取前后章节 |
| `hooks/useReadingProgress.ts` | 阅读进度：localStorage + Rust 后端双写阅读位置 |

## 漫画模块

| 文件 | 功能 |
|---|---|
| `components/MangaLibrary.tsx` | 漫画书架：系列列表、批量操作、图标管理、排序 |
| `components/MangaReader.tsx` | 漫画阅读器：图片流阅读、单页/双页/滚动模式、方向箭头 LTR/RTL |

## 公共组件

| 文件 | 功能 |
|---|---|
| `components/SharedUI.tsx` | 共享 UI：右键菜单、批量操作栏、复选框、收藏星标、进度条、图标选择器、排序按钮、返回按钮、顶部毛玻璃样式 |
| `components/FlatIcons.tsx` | SVG 图标库：全部扁平风格图标 ~50 个 |
| `components/ImportToast.tsx` | 导入进度弹窗：扫描/导入进度条、错误列表展开、done 事件自动关闭 |
| `components/ParticleCanvas.tsx` | 粒子背景动画：Canvas 粒子星空效果 |
| `components/DebugPanel.tsx` | 调试面板：前后端日志缓存与展示 |

## 工具/效果

| 文件 | 功能 |
|---|---|
| `utils/glow.ts` | 卡片聚光效：书架卡片鼠标跟随点光源 |

## Rust 后端（`src-tauri/src/`）

| 文件 | 功能 |
|---|---|
| `lib.rs` | Tauri 入口/路由：全部 command — 扫描/导入/分页/搜索/设置/收藏等 |
| `parser.rs` | 文件解析：TXT/EPUB/HTML 编码检测、章节切割、文本提取 |
| `comic.rs` | 漫画扫描：目录扫描、图片元数据读取 |
| `reader.rs` | PDF 渲染：mutool PDF→图片渲染、DPI 控制、LRU 缓存 |
