A minimalist reading app for immersive experiences. / 一款极简沉浸式阅读应用“墨读”
基于 Tauri v2 构建，底层用 Rust 编写，前端用 React + TypeScript。安装包 Lite 版不到 20 MB，Full 版（丰富字体包）约 200 MB。对比 Electron 动辄三四百 MB 的安装包，墨读启动更快、资源占用更少。
🔒 你的数据，只在你的电脑上
所有文件都是本地扫描、本地存储。没有账号注册，没有数据上传，没有远程服务器依赖。断网也能用——因为墨读本来就不需要网络。
🎯 精准到段落
传统的分页以"字符数"裁页，经常把一段话切到下一页。墨读用 DOM 测量分页引擎，读取每段文字的实际渲染高度，逐段贪心切分，保证每一页都是从段落开头开始、段落结尾结束。
🎨 颜值即正义
现代扁平化 SVG 图标、4 套主题（亮色/暗色/纸色/护眼）、底栏集成所有阅读控制、窗口控制常驻不折叠、右键菜单边界保护——每个细节都经过打磨。
